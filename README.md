# Weather Data

A beginner-friendly data analysis project that works with weather data to explore patterns, trends, and useful insights such as temperature changes, humidity levels, rainfall, or other atmospheric conditions.

## Overview

This project focuses on analyzing weather-related data using Python. It is designed to help understand how weather data can be collected, cleaned, explored, and visualized for better insights.

## Problem Statement

Weather data is often large, raw, and difficult to understand at first glance. The goal of this project is to process weather data and turn it into meaningful information through analysis and visualization.

## Dataset

This project uses weather-related data that may include the following features:

- Temperature
- Wind speed
- Date and time
- Location


## Features

- Reads and processes weather data
- Performs data cleaning and preprocessing
- Fetches the weather data of specific location using longitude and latitude
- Converts the raw json file to comma seperated value file.

## Tech Stack

- Python
- pandas
- Jupyter Notebook

## Project Workflow

1. Load the weather dataset
2. Inspect and clean the data
3. Handle missing or inconsistent values
4. Perform exploratory data analysis
5. Visualize key weather trends
6. Draw insights from the results


## Installation
